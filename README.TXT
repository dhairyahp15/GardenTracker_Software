PROJECT TITLE: SIMPLE GARDEN TRACKER SOFTWARE

PURPOSE OF PROJECT: Keeping track of plants using object orientation Java.

VERSION: 24 Feb 2021

AUTHOR: DHAIRYA PRAJAPATI

USER INSTRUCTIONS: The GardenTracker.java file contains all the features of the software, you can add the plants
to your garden, keep track of total plants, get the name of plants with or without repetition of same plants. It 
is a very basic tracking software which uses partially filled arrays.
